Возможные параметры:
-fx-fint-family: имя_шрифта // используемый шрифт
-fx-font-size: размер_шрифта // размер шрифта. Пример: 5px, 18px
-fx-backfround-color: цвет_фона // в любом формате, но программа генерирует в hex (web - формат)
-fx-focus-color: цвет_фокуса // по аналогии с фоном
-fx-text-fill: цвет_теста // по аналогии с фоном
-fx-border-color: цвет_рамки // по аналогии с фоном
-fx-border-radius: радиус_скругления_рамки // число, для задания округления у краёв рамки
-fx-background-radius: радиус_скругления_фона // число, для задания округления у краёв фона
-fx-border-width: толщина_рамки // число, толщина рамки в пикселах
-fx-padding: число число число число // четыре числа, задающие внутренние отступы. Числа идут в порядке:
// верх, право, низ, лево. Если отступ не нужен, можно в нужную позицию поставить 0
